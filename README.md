AlainPeters
===========

Pre processing of rule results


requirements
============

* compiler supporting C++11
* poco c++ libs


setup
=====

to make:
 ```$ make```

to make doc:
 ```$ make doc```

to clean:
 ```$ make clean```

to full clean:
 ```$ make raz```
